HamProto Drawers

A set of attributes that gives sliding control with three simple options




Guide

Requirements: Jquery

Their are three hooks to use with the proto drawer library. Selector, Toggle, and Direction. Using these three options you can create powerful drawers.

Selector

Use simple css selectors to select any element on the page.

Toggle

Toggle, In, or Out will allow you to make hooks that only open or close, or toggle back and forth.

Direction

Four directions are available: Up, Right, Down, Left. Use these explicitely on activator objects, or leave them off to allow the drawer to slide in whatever direction it currently is moving. The default if never specified will be to the right.

Events

A couple events have been provided for more versatility. The default event is a click, but you can also specify a mouseleave, mouseenter, or hover event in the attribute name itself.
